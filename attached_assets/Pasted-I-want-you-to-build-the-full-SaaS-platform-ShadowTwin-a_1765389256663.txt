I want you to build the full SaaS platform **ShadowTwin**, an AI-powered Cybersecurity Digital Twin system.

ShadowTwin will generate a full “Digital Security Twin” of any company’s assets (Web Applications, APIs, Cloud Infrastructure, Network Services) and run autonomous AI-driven security simulations, vulnerability discovery, and continuous monitoring.

Build the project in a fully structured, scalable, production-ready way.

===========================================
== 1. PROJECT ARCHITECTURE (TOP LEVEL) ==
===========================================

Create the following structure:

/backend
/frontend
/infra
/shared
/scripts
/docs

Use a monorepo architecture with TypeScript across all services.

================================
== 2. BACKEND REQUIREMENTS ==
================================

Stack:
- Node.js + Express + TypeScript
- PostgreSQL (Prisma ORM)
- Redis (Caching + Rate Limiting)
- JWT Auth (Access Token + Refresh Token)
- Zod validation
- Winston logging
- Helmet + CORS + Rate Limits
- Queue system (BullMQ) for scans
- Background worker for AI-based scanning engine

Backend Modules:

1. **Auth Module**
   - /auth/signup
   - /auth/login
   - /auth/refresh
   - /auth/logout
   - Password hashing (bcrypt)
   - Email verification (placeholder)

2. **Company Module**
   - /company/create
   - /company/update
   - /company/members

3. **Project Module**
   - /projects/create
   - /projects/list
   - /projects/delete

4. **Twin Engine Module**
   - /twin/init        → creates a digital twin instance
   - /twin/simulate    → runs attack simulations using the worker
   - /twin/scan        → full security scan (headers, TLS, API checks, CVE checks)
   - /twin/report      → returns a structured report

Models (Prisma):
- User
- Company
- Project
- DigitalTwin
- ScanJob
- ScanReport

Security:
- OWASP compliance
- Audit logs
- API key system for external integrations

Worker Tasks:
- Reconnaissance (HTTP, DNS, TLS)
- Basic vulnerability scanning
- API endpoint probing
- Placeholder AI logic for the Digital Twin intelligence

=====================================
